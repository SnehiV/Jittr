API Endpoints

  HTML API

    Root

        GET / - loads React web app

  JSON API

    Users

        POST /api/users
        PATCH /api/users

    Session

        POST /api/session
        DELETE /api/session
        GET /api/session

    CheckIns

        GET /api/checkIns
        POST /api/checkIns
        GET /api/checkIns/:id
        PATCH /api/checkIns/:id

    Friends

        GET /api/friends
        POST /api/friends
        GET /api/friends/:id
        DELETE /api/friends/:id

    Locations  (bonus)
        GET /api/locations
        POST /api/locations
        GET /api/locations/:id
        DELETE /api/locations/:id
